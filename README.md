# 📚 Library Management System (Spring Boot + JPA)

A simple web application built with **Spring Boot, Spring Data JPA, Thymeleaf, and MySQL** to manage books and authors.  
This project demonstrates the use of **ORM (Object-Relational Mapping)** using JPA with a Many-to-Many relationship.

---

## 🚀 Features

- ➕ Add Authors
- ➕ Add Books
- 🔗 Many-to-Many relationship between Books and Authors
- 📄 List all books and authors
- ⚡ Automatic database schema generation using Hibernate
- 🌐 Simple UI using Thymeleaf templates

---

## 🛠️ Technologies Used

- Java 17
- Spring Boot
- Spring Data JPA (Hibernate)
- Thymeleaf
- MySQL
- Maven
